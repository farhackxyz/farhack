# Getting started

It's time to start building on Farcaster! Over time, this hacking guide will grow to offer a variety of lessons and tutorials to help you grow your Farcaster development skills.

Apart from the [Building a Farcaster client tutorial](/docs/tutorials/litecast/overview), here are some other key resources that might be helpful!
- [Farcaster's Official Docs](https://docs.farcaster.xyz/)
- [Neynar](https://neynar.com): The easiest way to build on Farcaster
- [Airstack](https://airstack.xyz): The easiest way to build composable blockchain apps
- [frames.js](https://framesjs.org): fastest way to build Frames
- [Frog](https://frog.fm): Framework for Farcaster Frames
